# Scroll Progress Indicator

A simple, lightweight web application that visualizes reading progress. As the user scrolls down the page, a progress bar at the top of the viewport fills up to indicate how much of the content has been viewed.

## Features

- **Real-time Progress Tracking:** The progress bar updates instantly as you scroll.
- **Dynamic Content Generation:** JavaScript automatically generates sample text content to demonstrate the scrolling functionality.
- **Fixed Position:** The indicator remains visible at the top of the screen at all times.
